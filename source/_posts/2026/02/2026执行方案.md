# 一、多变量微积分（工程 / ML 定向方案）

> 定位一句话：
> **你学多变量微积分不是为了考试，而是为了“理解模型在干什么”**

---

## 1️⃣ 多变量微积分在你当前工作的真实用途

你已经在做 / 将要做的事情中，多变量微积分出现的位置：

| 场景              | 对应数学           |
| ----------------- | ------------------ |
| 多维 loss 优化    | 梯度、方向导数     |
| CNN / Transformer | Jacobian、链式法则 |
| 端侧量化 / 剪枝   | 局部敏感性         |
| 多模态对齐        | 高维空间几何       |
| LoRA / 微调       | 子空间投影         |

👉 **重点不是算题，而是“几何直觉 + 梯度传播”**

---

## 2️⃣ 多变量微积分学习范围（明确边界）

### **必须掌握（90% 工程足够）**

#### （一）多变量函数

* f(x, y, z) 的几何意义
* 等高线 / 等值面

#### （二）偏导数 & 梯度

* ∂f/∂x 的真实含义
* 梯度向量 = 最陡上升方向
* 梯度与 loss surface 的关系

#### （三）方向导数

* 为什么梯度方向最重要
* SGD 在干什么

#### （四）链式法则（核心）

* 多层网络中的梯度传播
* Jacobian 的“工程直觉版”

#### （五）多重积分（只要直觉）

* 体积 / 概率 / 期望
* 不需要手算复杂积分

---

### **可选（知道存在即可）**

* 拉格朗日乘子（约束优化）
* Hessian（只理解“曲率”）

## 4️⃣ 工程化学习方法（非常重要）

### ❌ 错误方式

* 做大量符号运算
* 追求完整证明

### ✅ 正确方式（你该用的）

每一节课只回答 3 个问题：

1. **几何上这是什么？**
2. **在神经网络里它对应什么？**
3. **如果没有它，会发生什么问题？**
